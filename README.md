This program analyzes the performance of students in a class. It accepts marks (out of 100) for up to 10 students, calculates the total and average marks, identifies the highest and lowest scores, counts pass and fail students, and generates a grade distribution summary (A+, A, B, C, D, Fail).
The program first asks the user for total number of students ( maximum 10 ) . Then , uses a 1D array , it stores the marks of each student . 
A loop calculates the total marks , highest , and lowest scores . By using if - else conditions , it clarifies each mark into a grade category ( A+ ,A ,B, C ,D, or Fail ) .
It also counts the number of passed students and also number of failed students .
The programs computees the average marks using total / no_of_students . And displays all the resulyts in  well - formatted summary , including grade distribution .
